Hotel Finder and Booking Application

Hotel Finder and Booking Application is a full-stack application that allows users to search for nearby hotels within a specified radius, view available rooms and prices, make bookings, manage their reservations, and leave feedback on their stay. Features

  • Search Nearby Hotels: The user specifies a radius in kilometers to find all the nearby hotels
  • List Hotels: The application lists and displays all hotels found
  • View Hotel Details: The user can select a specific hotel and see all available rooms alongside their prices
  • Book Rooms: The user can book one, or more of the available rooms
  • Manage Reservations: The application allows the user to cancel their reservation or change the booked room at least two hours before the check-in
  • Leave Feedback: User can leave a feedback

Tehnologii folosite:

  • Java Spring
  • HTML
  • CSS
  • React
  • MongoDb
